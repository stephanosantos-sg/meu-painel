Sincronizacao na Nuvem

Como configurar:
1. Crie uma Google Sheets nova
2. Renomeie a primeira aba para Data
3. Va em Extensoes > Apps Script
4. Apague tudo e cole o codigo abaixo:
function doGet() { var s = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Data'); return ContentService.createTextOutput(s.getRange('A1').getValue()) .setMimeType(ContentService.MimeType.JSON); } function doPost(e) { var s = SpreadsheetApp.getActiveSpreadsheet().getSheetByName('Data'); s.getRange('A1').setValue(e.postData.contents); return ContentService.createTextOutput('ok'); }
5. Clique Implantar > Nova implantacao
6. Tipo: App da Web
7. Executar como: Eu / Acesso: Qualquer pessoa
